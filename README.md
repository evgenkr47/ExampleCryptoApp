# Crypto
* О приложении:
  * Приложение состоит из 2х экрано(Первый - список криптовалют, Второй экран деталей)
  * Список с актуальными данными из списка обновляется каждые 10 секунд
  
* Технологии:
  * MVVM
  * Coroutines
  * Retrofit
  * Room
  * Dagger 2
  * Work Manager (обновляет данные каждые 10 секунд)
  * Picasso
